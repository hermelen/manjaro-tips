# Manjaro
## Global
- win+maj+q quitter un programme
- win+maj+r reload i3
- win+v = vertical mode (hpour horizontal)

## Terminal
- terminal ctrl D pour quitter
- maj + bas pour créer des "onglets"
- maj + droite ou gauche pour se déplacer

### FZF dans Terminal
- Alt + c => FZF (terminal search file)
